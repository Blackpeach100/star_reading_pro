# Star Reading 鑑定師ツール — GitHub Pages 配置手順

## 📁 必要なファイル一覧

以下の4ファイルを、すべて同じフォルダに置いてください：

```
star-reading/
├── index.html        ← star_reading_kanteishi_v24.html をリネーム
├── sw.js             ← Service Worker（オフライン対応）
├── manifest.json     ← PWAマニフェスト
├── icon-192.png      ← アプリアイコン（192×192px）※任意
└── icon-512.png      ← アプリアイコン（512×512px）※任意
```

> **アイコンについて：** なくても動作します。用意できれば
> ホーム画面に追加したときのアイコンになります。
> 金色や星のデザインがおすすめです。

---

## 🚀 GitHub Pages 公開手順（初回）

### 1. GitHubにリポジトリを作る

1. GitHub (github.com) にログイン
2. 右上の「＋」→「New repository」をクリック
3. Repository name に `star-reading`（何でもOK）と入力
4. **Private**（非公開）を選択 ✅ ← これ重要！
5. 「Create repository」をクリック

### 2. ファイルをアップロード

1. 作成したリポジトリのページを開く
2. 「uploading an existing file」リンクをクリック
3. 上記4ファイルをドラッグ＆ドロップ
4. 「Commit changes」をクリック

### 3. GitHub Pages を有効化

1. リポジトリの「Settings」タブを開く
2. 左メニューから「Pages」をクリック
3. Source を「Deploy from a branch」に設定
4. Branch を「main」「/(root)」に設定
5. 「Save」をクリック

### 4. URLの確認

数分後、以下のようなURLが表示されます：
```
https://あなたのユーザー名.github.io/star-reading/
```

このURLをiPhone/iPadのSafariで開いてください。

## 📱 iPhone・iPad でホーム画面に追加する手順（相手の方に伝える）

1. **Safari** でURLを開く（Chromeは不可）
2. 画面下の「**共有ボタン**」（四角に矢印のアイコン）をタップ
3. 「**ホーム画面に追加**」をタップ
4. 名前を確認して「追加」をタップ
5. ホーム画面に「Star Reading」アイコンが追加される
6. 次回からはアイコンをタップするだけで起動

> 一度開いた後は、**機内モードでも使用可能**です。

## 🔒 セキュリティについて

- リポジトリを **Private** にすることで、URLを知らない人はアクセスできません
- ただし、URLを知っていればアクセスできるため、URLの取り扱いにご注意ください
- コードの完全な保護は技術的に困難なため、**利用規約・NDA等の契約での保護**を併用することをおすすめします

## 🔄 ファイルを更新したいとき

1. GitHubのリポジトリページを開く
2. 更新したいファイルをクリック
3. 右上の鉛筆アイコン（Edit）をクリック
4. または新しいファイルをアップロード
5. 数分後に自動的に反映されます

## ❓ よくある質問

**Q. オフラインでも使えるの？**
A. 一度Safariで開いてホーム画面に追加すれば、その後はオフラインでも使えます。

**Q. データは保存される？**
A. 入力データはブラウザのローカルストレージに保存されます。アプリを削除すると消えます。

**Q. 複数の端末で使いたい**
A. 各端末でそれぞれ「ホーム画面に追加」してください。データの同期はされません。
